# SNOW DApp

Solana DApp for SNOW token ecosystem.

## Features
- SNOW Token Staking
- Frosty Swap AMM
- Iceberg DAO Governance
- Arctic NFTs

## Tech Stack
- Anchor 0.28.0
- React 18.2.0
- TypeScript 5.0.0
- Solana Web3.js 1.87.6

## Programs
- snow_dapp: SNOW staking and governance
- frosty_swap: AMM for SNOW pairs
- arctic_nfts: NFT marketplace
